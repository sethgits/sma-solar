Privacy Policy

Last updated: April 2, 2026

1. Data Collected

This application collects solar energy generation data (power output, energy totals, device metrics) from SMA inverters via the SMA Monitoring API. No personal data from any third party is collected.

2. Data Storage

All data is stored locally on the operator's private server in an InfluxDB time-series database. Data is never transmitted to external services beyond retrieval from the SMA API.

3. Data Sharing

No data is shared with any third party. This is a single-user private application.

4. Cookies & Tracking

This site uses no cookies, analytics, or tracking of any kind.
