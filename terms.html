1. Overview

SMA Solar Monitor is a private application used solely by its owner to monitor personal solar energy generation data via the SMA API. No third-party users are permitted.

2. Data Use

Solar generation data retrieved from the SMA API is stored in a private InfluxDB instance on the owner's own hardware and is never shared with third parties.

3. Access

This application accesses only data for plants owned by the application operator. No other users' data is accessed or stored.
